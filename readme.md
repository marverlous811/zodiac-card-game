# Description

## Requirement

- node version >= 8.10.0

- npm version >= 5.6.0

- typescript 

## Usage

- install

    `npm install`

- Config

    `cp config.temp config.ts`

- start
    
    `npm run start-server`

- test core

    `npm run start-test`

- test server

    `tsc && npm run api-test`
